One-Time Backend Setup (maintainer only)
  1. Go to console.firebase.google.com and create a project (any name, Analytics off).
  2. Open Build > Authentication > Get started and enable Email/Password sign-in method.
  3. Open Build > Firestore Database > Create database, choose Production mode and region.
  4. In the Rules tab, paste rules (button below) and click Publish.
  5. Open Project settings (gear icon): copy Project ID and Web App apiKey.
  6. Paste both into src/firebase-sync.js, run node build.js, and push.
